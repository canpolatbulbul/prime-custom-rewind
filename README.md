# Prime Video Custom Skip

A lightweight Chrome/Firefox browser extension built using Manifest V3 that introduces precise, customizable fast-forward and rewind controls for Amazon Prime Video. 

This extension overrides native 10-second skip boundaries, allowing you to fine-tune skips (e.g., down to 2–3 seconds) so you can easily skip or repeat dialogue without losing the context of the closed captions.

## ✨ Features

- **Custom Skip Times:** Easily configure exact rewind and forward seconds via a clean popup interface.
- **Capturing Phase Overrides:** Overrides the default Amazon player shortcut listeners gracefully.
- **Form-Safe:** Automatically disables key tracking while typing in search inputs or text boxes.

## ⌨️ Shortcuts

- **Rewind Control**: Default is <kbd>,</kbd> or <kbd><</kbd> (Fully customizable)
- **Fast-Forward Control**: Default is <kbd>.</kbd> or <kbd>></kbd> (Fully customizable)

*To remap shortcuts, click directly on the extension icon badge, click the desired key configuration input box, and press your preferred key assignment.*

## 🛠️ Local Installation (Developer Mode)

1. Clone or download this repository to your local machine.
2. Open a Chromium-based browser (Chrome, Edge, Brave, Opera) and navigate to `chrome://extensions/`.
3. Enable **Developer mode** using the toggle switch in the top-right corner.
4. Click **Load unpacked** in the top-left corner.
5. Select the folder containing these extension files.
